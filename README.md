## Folder Structure

The files contain multiple data structures, in their static and dynamic versions.:

- `src`: the folder to maintain sources
- `lineales`: contains lineal data structures.
- `dinamicas`: contains the structures in their dynamic versions
- `estaticas`: contains the structures in their static versions
- `tests`: contains various JUnit tests and other simple tests used during development.
- `lib`: the folder to maintain dependencies

Meanwhile, the compiled output files will be generated in the `bin` folder by default.
